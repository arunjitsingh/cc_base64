Base64 in C++

A basic Base64 encoder-decoder in C++.

Test cases require googletest, although there is some custom Scons scripts that build that in, so you won't be able to run the tests directly. You can use your own makefile/ to create a library for base64 and gtest and link them to the test binaries.

You can raise bugs on this project's issue tracker.
